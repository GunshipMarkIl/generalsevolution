# Command &amp; Conquer: Generals Evolution
![alt text](https://github.com/GunshipMarkIl/generalsevolution/blob/main/topdown.2.webp?raw=true)

The goal of Generals Evolution is to re-imagine C&C Generals on Red Alert 3 engine, in other words it's a total conversion of Red Alert 3, which features better online play, better graphics fidelity and stability. Generals Evolution also bring new units, overhauled visuals, new maps, new sounds and game feel. Please note that it's not 1:1 port of C&C Generals and it's not meant to replace original game, but rather imitate it as close as it is possible.

- Make sure to launch Red Alert 3 at least once before playing the mod.
- Make sure to install Tacitus(d3d9.dll), located in the Tacitus folder, into your game Data folder.
- Please note that there is no campaign currently in the mod, only skirmish and online play, loading campaign will crash the game.
- Please note that your antivirus can block/remove GenEvoLauncher.exe, add an exception, this is a false positive.
- Please completely uninstall old version of the mod, do not install B0.31 over 0.1/0.2/0.21/0.3/0.31, do complete clean install of the mod.
- If this is your first time installing this mod and you don't own Red Alert 3 then please make sure that you're buying Red Alert 3 and not Red Alert 3 Uprising. Red Alert 3 Uprising doesn't support modding, buy Red Alert 3 instead.

    Make sure to launch Red Alert 3 at least once on your system, tweak graphical and resolution settings to your liking.
    Unzip contents of the archive.
    Navigate to your Mods C:\Users\[USER NAME]\Documents\Red Alert 3\Mods\ folder, create it there if you don't have it.
    If you had previous installations of GenEvo - remove them by completely removing GenEvo folder from your Mods folder.
    Put GenEvo folder into your Mods folder.
    Launch the mod using GenEvoLauncher.exe inside GenEvo folder or via -ui launch parameter.
    Enjoy!
